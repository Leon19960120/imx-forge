关键点验证:
-----------
1. 设备树节点匹配成功 (/imx_aes_led)
2. 寄存器地址映射成功
3. GPIO1_IO03 配置为输出模式 (GPIO1_GDIR = 0x108, bit 3 = 1)
4. LED控制逻辑正确:
   - 写 '1': GPIO1_DR bit 3 = 0 (低电平点亮)
   - 写 '0': GPIO1_DR bit 3 = 1 (高电平熄灭)
5. 字符设备 /dev/AES_LED 正常工作
6. 卸载时正确释放资源
