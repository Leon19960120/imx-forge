# 欢迎来到IMX-Forge的教程文档

嘿！这里是**IMX-Forge**的嵌入式Linux教程文档，随意看看！本工作室专注于IMX6ULL平台的嵌入式开发，从零开始带你构建完整的嵌入式Linux系统。

目前主要有以下几个部分组成：

- **工具链准备** - 从工具链开始，搭建你的开发环境
- **U-Boot教程** - 深入理解Bootloader，学会移植和调试
- **内核教程** - Linux内核编译、配置与设备树
- **根文件系统** - 构建你自己的嵌入式根文件系统
- **实战演练** - 整合所有知识，完成一个完整的项目
